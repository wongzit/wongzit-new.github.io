py.Aroma 5 is a free, open-source program toolkit for aromaticity analysis. Optional functionalities for plotting NMR spectra, a computational supporting-information generator, and geometry operations are integrated.

What it does

🧭
NICS
Single-point, 2D, 3D, XY-Scan & integrated NICS
🪞
Dissected NICS
NICSπ via the NBO program
📐
HOMA / HOMER
Geometric aromaticity indices
📊
BLA
Bond-length alternation with color maps
🧊
POAV
π-orbital axis vector analysis
⚛️
NRT
Natural resonance theory analysis
📈
NMR plotting
Generate publication-ready spectra
🗂️
SI generator
Computational supporting information
🔧
Geometry ops
Various operations on geometry

Built with

Python 3.13 PyQt6 Matplotlib NumPy OpenPyXL NetworkX
