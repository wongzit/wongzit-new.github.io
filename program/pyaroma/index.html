py.Aroma

A free, open-source toolkit for aromaticity analysis.

🏕️ home/ ⬇️ download/ 📄 manual/ 💎 citations/ ⚖️ license

py.Aroma 5 is a free, open-source program toolkit for aromaticity analysis. It supports BLA, HOMA, HOMER, POAV, NICS (Single point, 2D, 3D, XY-Scan, Integrated NICS), dissected NICS (i.e., NICSπ via the NBO program), and natural resonance theory (NRT) analysis. Optional functionalities for plotting NMR spectra, a computational supporting-information generator, and geometry operations have been integrated.

py.Aroma 5 is built with Python 3.13 and a PyQt6 graphical user interface. It uses third-party libraries including Matplotlib, NumPy, OpenPyXL and NetworkX. This project is released under the CC BY-NC-SA 4.0 license.

To download py.Aroma 5, please visit the download page. For the program user manual, please check the manual. Known issues in the latest version are summarized on the issues page — if you find a problem, please refer to it or contact me.

Feature requests and feedback are welcome. Feel free to contact me via e-mail, or use the Contact Developer button in the start-up window of py.Aroma 5. Questions and bug reports are appreciated.
